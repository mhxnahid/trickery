#### Basic commands
```
Save - :w
Save and quit - :wq
Quit - :q
Quit without saving - :q!
```
#### Copy/Paste
```
Press `Shift+v` to enter visual mode. Mark lines and press `y` to yank. In command mode, press `p` to paste.

Copy to register:
Go to visual mode, mark lines, press `"ay` to yank into register `a`, back to normal mode, press `"ap` to paste.

Copy to system clipboard:
xclip has to be installed. Go to visual mode, mark lines, press `"+y` to yank into clipboard.
```
#### Undo
```
Press `u`
```
